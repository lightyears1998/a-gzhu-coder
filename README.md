# 仓库结构

仓库中的文件为UTF-8编码，使用Windows风格的换行符（“\r\n”）。

- **acm** ACM-IPCP相关内容
- **book** 有趣/有用的书
- **camo** 图片仓库
- **howto** F.A.Q.
- **pre-school** 新生入学前的学姐学长布置的课程作业
- **snippets** 代码片段
- **utils** 小工具
