# Graph工具与计算机图形学

与计算机图像相关的笔记

## 基本概念

- **图像技术** 造型技术、光照模型和渲染（绘制）技术
- **帧缓冲器** 对帧缓冲器的写入等同于对屏幕的写入
- **光栅图形** 与 **矢量图形**

## 光栅图形学算法

### 直线扫描转换算法

- 朴素算法 斜截式方程取点四舍五入 `y=kx+b`
- 增量算法改进斜截式方程 数值微分算法DDA（Digital Difference Analyzer） `x(i+1)=x(i)+1, y(i+1)=y(i)+k`
  |k|<1时较为合适；当|k|>1时出光栅点过于稀疏的情况
- 中点画线算法
