## User Interface Basic

Viewport Navigation

- *Alt + 左键* 视角
- *Alt + 中键* 摄像机位置
- *Alt + 右键* 远近
- *F / A* 聚焦或全部显示
- *Space* 分镜头 / 快捷菜单
- *长按右键* 上下文快捷菜单

工具和窗口的快捷键

- *W, E, R* 移动，旋转和缩放工具
- *Ctrl + A* 属性窗口
